Gemini API キーの取得手順

Gemini APIキーは、Google公式の開発者ポータルである Google AI Studio から数ステップで簡単に、かつ完全無料で発行できます。

1

Google AI Studio へアクセス

下記の公式リンクをクリックして、お使いのGoogleアカウントでサインインします。

Google AI Studio 公式サイトへ
2

「鍵マーク（Get API key）」をクリック

画面の左端メニュー（サイドバー）の一番下あたりにある「鍵の絵のアイコン（Get API key）」をクリックします。

3

キーを生成してコピーする

「Create API key」を選択し、任意のプロジェクトを選択するか新規作成（Create API key in new project）を選択します。数秒で AIzaSy... から始まる文字列が生成されますので、これをコピーします。

4

アプリの入力欄に貼り付け

コピーしたキーを、このTTSプレイヤー画面右上、またはヘッダーにある「APIキー入力欄」に貼り付け、「保存」ボタンを押してください。

安全性について

入力したAPIキーは、外部のいかなるサーバーにも送信されません。あなたのブラウザのローカルメモリ（LocalStorage）にのみ安全に保存され、Gemini APIとの通信に直接使用されます。
